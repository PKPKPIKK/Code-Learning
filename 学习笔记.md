# 3.字面量
    1.整数：int a=10;
    2.小数：double a=10.0;
    3.字符：char a='a'; (单引号，内容只有一个)
    4.布尔：boolean a=true;
    5.字符串：String a="hello world";   
    6.空类型：null

# 4.变量（为存储数据的小空间，不是数据）
    1.int a = 10;
      数据类型  变量名  =  变量值
    2.变量命名规则：
        1.只能使用字母、数字、下划线、$
        2.不能以数字开头
        3.不能使用关键字
    3.注意事项：
        1.变量名必须唯一
        2.变量只能存一个值
        3.变量在使用之前必须赋值
        4.一条语句可以创建多个变量，可以连续赋值
